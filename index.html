mongoFS

An enhanced file storage implementation in MongoDB that extends the GridFS functionality

About mongoFS

This project came out of a need in my company to extend the current GridFS functionality to include compression of file data. After spending a day or so trying to work with the existing implementation. I determined that the existing implementation was not going to be extendible due to the current use of inner classes and heavy cross-class collusion. So I decided to re-write the existing implementation in order to extend it.

This project is currently under development, stay tuned. I have version 0.6.1 out and available. Goto the downloads section to see how to get it. Check out my task list under the TODO to check the progress. I need to have this code into production by end of March so keep checking back to see the progress.
