Configuring to run mongoFS

The constructor requires two parameters, a com.mongodb.DB and me.lightspeed7.mongofs.MongoFileStoreConfig

For the DB object, the library leaves the task of connect to a database to you. However you choose to provide the host, port, database and credentials is up to you. You just need to provide the constructor with a com.mongodb.DB object setup and ready to go. This allows the maximum flexabilility within users systems.
